## Repository Snapshot
- `bin/` — Main executable (`usb-driver`).
- `lib/` — Shared libraries for mounting, verification, and UI.
- `scripts/` — Installation and uninstallation wizards.
- `docs/` — Manual documentation and tutorials.
- `config/` — Default configuration templates.
