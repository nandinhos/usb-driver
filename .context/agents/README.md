# Agent Playbooks

This directory contains specialized playbooks for AI agents working on the `usb-driver` project. These playbooks provide context-specific instructions, best practices, and resource links.

## Active Agents

- [Architect Specialist](./architect-specialist.md): Core structural design and bridge logic.
- [Bug Fixer](./bug-fixer.md): Hardware edge cases and attachment troubleshooting.
- [DevOps Specialist](./devops-specialist.md): Installation, environment checks, and cross-platform dependencies.

## Usage

When engaging an AI agent, refer them to the corresponding playbook in this directory to ensure they follow project-specific standards.
