# usb-driver

Ferramenta modular para automação de backup em pendrives EXT4 no WSL2.

## Estrutura
- `bin/`: Executável principal
- `lib/`: Módulos de lógica (checks, mount, usbipd)
- `config/`: Arquivos de configuração
- `scripts/`: Instalador e desinstalador

## Instalação
Consulte `scripts/install.sh`.
